O que é o MongoDB

Titulo Teste

Em vez de seguir o modelo de banco de dados relacional tradicional, que organiza os dados em tabelas com esquemas predefinidos, o MongoDB adota um modelo de dados flexível baseado em documentos. Assim, o MongoDB armazena os dados no formato BSON (Binary JSON), que permite estruturas de dados hierárquicas e não requer um esquema fixo.
